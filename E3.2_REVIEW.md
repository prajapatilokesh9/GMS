## 1. Scope Delivered

### Database Changes

- **Table**: `personal_training.pt_packages` (schema: `personal_training`)
- **Migration**: `20260621_sprint5_pt_packages` (manual SQL, 24 lines)

| Column | Type | Constraints |
|---|---|---|
| `id` | UUID | PK |
| `tenant_id` | UUID | FK → `public.tenants`, NOT NULL |
| `gym_id` | UUID | FK → `gym.gyms`, NULLABLE |
| `name` | VARCHAR(200) | NOT NULL |
| `description` | TEXT | NULLABLE |
| `total_sessions` | INTEGER | NOT NULL |
| `price` | DECIMAL(10,2) | NOT NULL |
| `validity_days` | INTEGER | NOT NULL |
| `is_active` | BOOLEAN | DEFAULT true |
| `created_at` | TIMESTAMP(3) | DEFAULT now() |
| `updated_at` | TIMESTAMP(3) | @updatedAt |

**Indexes**: `@@unique([tenantId, name])`, `@@index([tenantId, isActive])`, `@@index([tenantId, gymId])`

**Relation**: `PtPackage` has optional `gym` relation → `gym.gyms`; cascade delete on tenant.

### API Endpoints

All routes mounted at `/api/v1/pt-packages`, protected by `authenticate` + `loadPermissions` middleware:

| Method | Path | Auth | Description |
|---|---|---|---|
| `GET` | `/api/v1/pt-packages` | `pt-package:read` | List packages (optional `gymId`, `isActive` filters) |
| `POST` | `/api/v1/pt-packages` | `pt-package:manage` | Create package (Zod validation) |
| `GET` | `/api/v1/pt-packages/:id` | `pt-package:read` | Get package by ID |
| `PATCH` | `/api/v1/pt-packages/:id` | `pt-package:manage` | Update package fields |

### Permissions

Seeded in `src/database/seed.ts`:

- `pt-package:read` — category `personal_training`
- `pt-package:manage` — category `personal_training`

### Events

Published via `publishBillingEvent` to the `billing` BullMQ queue:

| Event | Trigger | Payload |
|---|---|---|
| `pt.package.created` | POST create | `{ packageId, name }` |
| `pt.package.updated` | PATCH update | `{ packageId, changes }` |

### Admin UI

Three pages at `/admin/pt-packages/`:

| Route | File | Description |
|---|---|---|
| `/admin/pt-packages` | `page.tsx` | List all packages |
| `/admin/pt-packages/new` | `new/page.tsx` | Create package form |
| `/admin/pt-packages/[id]` | `[id]/page.tsx` | Package detail/edit |

Frontend API layer: `src/lib/api.ts` (lines 283–305) — `listPtPackages`, `getPtPackage`, `createPtPackage`, `updatePtPackage`.
